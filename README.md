## Overview
This interactive Tableau dashboard showcases sales performance across regions, products, and channels for 2022-2024. It highlights key metrics such as Total Sales, Total Profit, Average Profit Ratio, and Total Orders, while enabling dynamic exploration via filters for Year and Region.
The dashboard includes advanced visualizations: Big Number KPI cards, Monthly Trends Line Chart with average line, a Supplier Performance Scatter Plot with Profit Ratio categorization, and Product Category TreeMaps. Payment channels and types are analyzed through Pie Charts. 
Designed with a clear color scheme, intuitive layout, and interactive tooltips, this dashboard demonstrates my ability to translate complex data into actionable insights and visually compelling storytelling.
